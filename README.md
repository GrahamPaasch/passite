# passist
passing siteswap assistant

https://passist.org/

Work in progress..

# Features
- Siteswap generator
- Start configuration calculator
- Causal Diagrams
- Web App that works offline
- Reusable js components

# planned
- 3d Animation
- API
- Causal Diagram Editor


Usage:
```
 npm install
 npm build
 npm start
 open http://localhost:3000
```
